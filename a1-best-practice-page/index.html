The purpose of lazy loading is not to load resources or images immediately but rather load them later on when needed. This technique makes web pages load faster as the loading reduces the number of images to be loaded right away up within the page.

As a website administrator, you can also save up money by using this technique rather than relying on a bunch of cloud hosting services such as Content Delivery Networks (CDN), storages, and etc.
